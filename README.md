Node.js Weather App - Accepts city, returns Temperature in Fahrenheit.

Dependencies:

	npm install --save 

		ejs
		
		request

		express

		yargs

		Body-Parser
		
	OpenWeatherMap.org API key

Runs on Port 3000 Currently
